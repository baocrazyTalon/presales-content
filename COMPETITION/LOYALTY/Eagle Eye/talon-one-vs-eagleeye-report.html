Executive
Summary

Eagle Eye AIR started as a loyalty wallet engine. Promotions were retrofitted as an add-on called "Cloud Basket Adjudication" — a separate product, a separate POS-specific API, and a separate set of templated offer types that cannot be customised without overriding rules in the request payload (i.e. coding). The split means checkout flows require multiple API calls across two systems, offer types are fixed and not extendable by marketers, and features like gamification, strikethrough pricing, SDKs, and open documentation are either absent or acquired externally. Talon.One resolves all of this from a single rule engine and a single API call — decisioning promotions, loyalty, coupons, and gamification atomically, at sub-60ms, without code for every new use case.

API Design & Checkout Latency

Eagle Eye's split between the Loyalty Transaction API and the POS Connect (promotion adjudication) API creates a fundamental checkout complexity problem — one that Talon.One's unified Session API eliminates by design.

Eagle Eye AIR — Checkout Flow

1Authenticate customer / open wallet (POS Connect: /connect/wallet/open)
2Retrieve targeted offers from wallet (Loyalty API: wallet accounts / coupons)
3Send basket to adjudication with options flags (analyseBasket, adjustBasket, basketAdjudication — each a separate option in the payload)
4Receive POS Connect response with discount result
5Separately calculate loyalty points via Transaction API (or settle later in batch)
6Merge discount result + points result in your own middleware before presenting to customer
7Settle transaction to commit points (can be deferred — async latency risk)
5–7 calls across 2 separate APIs + custom middleware merge required

Talon.One — Checkout Flow

1Call UpdateSession with cart contents, customer ID, and any coupon codes
2Receive single atomic response: applied discounts, points earned, coupon validity, gamification progress, messages
3Close session on order confirmation — atomic commit of all effects
1 call · 1 API · full incentive result · no middleware

Where Eagle Eye Is Genuinely Competitive Eagle Eye's strongest capabilities are in the loyalty wallet core: multi-ledger architecture, standard earn/burn templates, Stampcard, Bounceback, Continuity, and Meal Deal bundle pricing. Conclusion & Recommendations

Eagle Eye AIR is a loyalty-first platform that added promotions as a bolt-on. The result is two separate API surfaces, slower checkout latency, templated offer types that cannot be customised without coding, and missing capabilities — strikethrough pricing, gift cards, SDKs, public documentation, native gamification, B2B cart handling — that modern retail requires. Combined with a modular pricing model designed to expand contract value 3× over 5 years, the total cost of working around Eagle Eye's gaps consistently exceeds Talon.One's all-in licence.

Talon.One resolves all of this from a single rule engine and a single API call — decisioning every incentive type atomically in 40–60ms, configurable by marketers without code, with documented APIs, SDKs, MACH certification, and flat pricing for the full scope.

Openness & Future-Proofing
