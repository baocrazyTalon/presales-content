API

Integration API

The Integration API is the primary interface between your system and Talon.One. It evaluates promotions and loyalty rules in real time using a session-based model.

Session Flow

Your SystemCart / POS event
→
PUT /v2/customer_sessions/{id}Integration API
→
Rule Engine<50ms evaluation
→
EffectsDiscount, points, gift

Key Endpoints

MethodEndpointPurpose
PUT/v2/customer_sessions/{integrationId}Open / update a customer session
PUT/v2/customer_profiles/{integrationId}Update customer profile attributes
GET/v1/loyalty_programs/{id}/profile/{id}Read loyalty ledger balance

Effect Types

💸

setDiscount

Apply a percentage or fixed-amount discount to the session total or a specific line item.

⭐

addLoyaltyPoints

Credit points to the customer's loyalty ledger upon qualifying purchase or event.

🎁

giveGift

Grant a free SKU or coupon code as a campaign reward.

Data

Product Catalog Sync

Keep the Talon.One SKU catalog in sync with your product feed to enable catalog-aware promotions (e.g. "10% off all items tagged as protein").

🔄

Incremental Sync

Push only changed SKUs on a scheduled or event-driven basis — no full catalogue reload required.

🏷️

SKU Attribute Mapping

Map product metadata (category, brand, tags) to Talon.One catalog item attributes for rule conditions.

🎯

Catalog-Based Rules

Write rules that target specific product categories, brands, or custom attributes without hardcoding SKUs.

Integrations

Third-Party Integrations

Talon.One connects to your existing marketing stack via webhooks and native connectors.

CDP / CEP (e.g. Braze, Customer.io, Klaviyo)

Trigger personalised communications on loyalty tier changes, reward issuance, and campaign milestones using Talon.One outbound webhooks.

Key events: loyalty.tier.changed, coupon.created, reward.redeemed, session.closed

Segmentation (e.g. Segment, Amplitude)

Sync audience segments and behavioural cohorts into Talon.One as customer attributes for hyper-targeted campaign conditions.

Architecture

Security

🔐

Encryption

TLS 1.2+ in transit. AES-256 at rest. All API keys are hashed and never returned in plaintext after creation.

🗝️

API Key Management

Scoped keys (read-only / full-access). Key rotation supported at any time without downtime.

🛡️

Compliance

SOC 2 Type II (available under NDA). GDPR controls — right to erasure, data portability, consent flags.

🌍

Data Residency

EU and US data residency options. No cross-region data transfer without explicit configuration.

Architecture

Hosting & Infrastructure

☁️

AWS Multi-Region

Deployed across multiple AWS availability zones for high availability and automatic failover.

📈

SLA & Performance

99.9% uptime SLA. Rule evaluation at <50ms p95. Up to 10,000 API requests per minute per integration.

💾

Backups & Recovery

Daily automated backups with point-in-time recovery. Disaster recovery RTO < 4 hours.
